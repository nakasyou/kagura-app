# KaguraJS + Vite + Vue
This project inited from [create-kagura](https://github.com/nakasyou/create-kagura)

This project uses [Vite](https://vitejs.dev) and TypeScript.

## How to develop this?
First, Install deps.
```bash
npm install # npm
yarn install # yarn
pnpm install # pnpm
```
Second, start dev server
```bash
npm run dev # npm
yarn dev # yarn
pnpm dev # pnpm
```
## How to build this?
```bash
npm run build # npm
yarn build # yarn
pnpm build # pnpm
```
If you type they commands, Vite builds static HTML files.
